# VIT Mobile

**VIT Network mobile client** — Telegram Mini App and standalone iOS/Android app.

[![React Native](https://img.shields.io/badge/React_Native-0.74-61DAFB?style=flat-square&logo=react)](https://reactnative.dev)
[![Telegram](https://img.shields.io/badge/Telegram-Mini_App-26A5E4?style=flat-square&logo=telegram)](https://core.telegram.org/bots/webapps)

## Features

- 🏆 Live match predictions and VIT scores
- 💳 Multi-currency wallet (NGN/USD/USDT/VITCoin)
- 🤖 AI intelligence feed from the agent swarm
- 🏪 Signal marketplace access
- 📊 Personal performance analytics

## Telegram Mini App

The Mini App runs inside Telegram and authenticates via `initData` verified server-side. Launch via the VIT Telegram bot.
